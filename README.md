## Tính năngMô tả ngắn:



- Đăng ký/Đăng nhập tài khoản- Mục tiêu: demo trang xem phim đơn giản.

- Xem danh sách phim- Cấu trúc: các file HTML ở thư mục gốc.

- Tìm kiếm phim

- Xem chi tiết phimHướng dẫn nhanh:

- Đăng tải phim (yêu cầu đăng nhập)

- Phát video trực tuyến1. Mở `index.html` hoặc `web.html` trong trình duyệt.

- Thống kê lượt xem2. Để phát triển: chỉnh sửa file, commit và push lên GitHub.
